# 夜半偷鹹魚

夜半偷鹹魚是一个面向手机的小说阅读与听书 PWA。这个分支是大陆听书版：纯静态部署、不依赖外部 CDN 或在线 TTS 服务，小说文本、封面和朗读控制都随网站一起发布。

## 功能

- 多书库：当前内置《神话培育流》和《勇者派送中》
- 简体中文界面与书稿
- 普通话 / 简体中文系统语音优先
- `标准`、`听书`、`戏剧` 三档朗读模式
- 书本分类、章节目录、搜索、书架、书签
- 播放、暂停、停止、上一章、下一章、快进、倒退
- 自动连播、睡眠定时、本章播完停止
- 防黑屏播放、锁屏媒体控制
- 阅读进度与偏好本机保存
- PWA 安装与离线缓存

## 本地预览

```powershell
powershell -ExecutionPolicy Bypass -File .\serve.ps1
```

电脑打开：

```text
http://localhost:5173/
```

手机和电脑在同一个 Wi-Fi 时，启动脚本会列出手机可访问的网址。

## 部署

这是纯静态网站，可直接部署到 GitHub Pages。
